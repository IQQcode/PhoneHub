## Phone Store

**基于移动端的手机商城**

**基于 SpringBoot + Vue 的前后端分离开发架构**



## 前端技术栈

**Vue + Vant UI + less**

Vant UI 安装命令

```shell
cnpm i vant -S
```

less安装命令

```shell
cnpm install less less-loader --save
```

快速启动

```shell
npm run serve
```



## 后端技术栈

**SpringBoot + SpringData JPA**



## 数据库

**MySQL**
