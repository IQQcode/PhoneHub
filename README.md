## 项目简介

**手机极客消息站，购机必备。让你轻松选购自己满意的高性价比手机📲**

在线上购买手机，给了你更多选择的机会，更方便的购买流程。但是，由于信息的不对称性，面对种类繁多、越来越同质化的智能手机，往往我们在购买时会陷入选择与纠结。

PhoneHub汇聚了手机的分类信息，直观的信息对比，新机预购，便捷的选购流程，让你的购机之旅便捷而满意。

**项目文档📚：[https://iqqcodes.gitee.io/coan](https://iqqcodes.gitee.io/coan)**

## 功能实现

- 手机特点展示
- 配置直观对比，手机配色对比，手机规格对比，手机价格对比

- Dxo排行查看
- 新机预览，
- 快捷下单，订单信息查看

## 技术栈

### 前端技术栈

**Vue + Vant UI + less**

Vant UI 安装命令

```shell
cnpm i vant -S
```

less安装命令

```shell
cnpm install less less-loader --save
```

快速启动

```shell
npm run serve
```

### 后端技术栈

**SpringBoot + SpringData JPA**

### 数据库

**MySQL**
